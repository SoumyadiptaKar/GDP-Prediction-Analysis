About the Project

This GDP Analytics Platform is a comprehensive web application developed as part of academic coursework at the University of Helsinki. The project demonstrates advanced data science, machine learning, and web development techniques applied to economic data analysis.

The platform provides interactive visualizations, machine learning predictions, and comprehensive analysis tools for exploring global economic indicators and their relationships.

Technologies Used:
  • Python (Flask, Pandas, Scikit-learn)
  • JavaScript (Plotly.js, Bootstrap)
  • SQLite Database
  • Interactive Data Visualization
  • Machine Learning Models
